##Features:

* Print any month 1-12 of any year 1-9999 when given following month/year input types:

 * April <year>
 * april <year>
 * Apr\* <year>
 * apr\* <year>
 *     4 <year>
 *    04 <year>
 *   004 <year>

* Print any year from 1-9999 when given input type:

 * 2013
 *    1
 *   01
 *  001
 * 0001
